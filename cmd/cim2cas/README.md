# cim2cas

Convert a .cim file which zmac outputed to a .cas file which fMAX can load.

zmac が出力した .cim ファイルを fMSX 等で読める .cas ファイルに変換する。

## How to use

```
cim2cas -cim {INPUT} -cas {OUTPUT} [-nam {NAME_IN_TAPE}] [-off {OFFSET}]
```
